Estrutura Base do HTML

Veja como é a estrutura mínima de um documento HTML:


<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <title>Título</title>
</head>
<body>
    Conteúdo visível
</body>
</html>

<head> → Contém metadados.
<body> → Contém o conteúdo visível.

Representação de Dados

  • Bit → 0 ou 1
  • Byte → 8 bits
  • UTF-8 → código multibyte
  • 1024 bytes = 1 KB
  • 1024 KB = 1 MB
  • 1024 MB = 1 GB

MB (armazenamento) ≠ Mb (transmissão)

Semântica no HTML5

Negrito

Forma visual (não semântica):


<b>Texto em negrito</b>

Significado semântico:


<strong>Texto importante</strong>

Itálico


<i>Texto em itálico</i>


<em>Texto com ênfase semântica</em>

Tipos de Listas

Lista Ordenada


<ol type="I">
  <li>Item 1</li>
  <li>Item 2</li>
</ol>

Lista Não Ordenada


<ul>
  <li>Item</li>
  <li>Item</li>
</ul>

Lista de Definição


<dl>
  <dt>HTML</dt>
  <dd>É sobre significado.</dd>
</dl>

QUEBRA DE LINHA, EMOJIS E SÍMBOLOS


<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <title>Parágrafos</title>
</head>
<body>
    <h1>Parágrafo e quebra de linha</h1>
    <hr>

    <p>Pode ser usado:</p><br>

    <p>&reg; - marca registrada</p>
    <p>&copy; - copyright</p>
    <p>&trade; - trademark</p>
    <p>&yen; - moeda japonesa</p>
    <p>&euro; - euro</p>

    <hr>

    <h1>Vamos colocar emojis agora</h1>
    <hr>

    <p>&#x1F917;</p>

</body>
</html>

<br> → quebra de linha
<hr> → linha horizontal
