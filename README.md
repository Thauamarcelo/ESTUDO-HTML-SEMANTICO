# 🧠 Página 1 – HTML Semântico

A primeira página demonstra o uso correto das principais tags semânticas do HTML5:

- `<header>`
- `<nav>`
- `<main>`
- `<section>`
- `<article>`
- `<aside>`
- `<footer>`

O objetivo foi estruturar o conteúdo de forma organizada e profissional, respeitando hierarquia de títulos e boas práticas.
